# 🧠 Sentiment Analysis Web App

A simple **Machine Learning app** that predicts whether a text is **positive, negative, or neutral** using a Logistic Regression model trained on social data.

## 🚀 Features
- Clean Streamlit UI
- Real ML model (TF-IDF + Logistic Regression)
- 80%+ accuracy
- Easy to run locally


💻 Tech Stack

Python

Streamlit

Scikit-learn

Joblib
